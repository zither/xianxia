# 仙途 (XianTu) - 修仙挂机游戏

## 项目简介

一款移动端网页挂机游戏，玩家扮演一名修仙者，通过修炼功法、闯荡副本、收集资源，逐步提升境界，最终渡劫成仙。

## 技术栈

- 纯原生 HTML/CSS/JavaScript
- 单文件应用，方便部署
- LocalStorage 数据持久化

## 核心系统

1. **境界体系** - 炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫
2. **修炼系统** - 挂机自动获取修为
3. **功法系统** - 主动/被动技能
4. **战斗系统** - 副本闯关
5. **资源系统** - 灵气、灵草、矿石
