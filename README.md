# 仙途 (XianTu) - 修仙挂机游戏

## 项目简介

一款移动端网页挂机游戏，玩家扮演一名修仙者，通过修炼功法、闯荡副本、收集资源，逐步提升境界，最终渡劫成仙。

## 技术栈

- 纯原生 HTML/CSS/JavaScript
- 单文件应用，方便部署
- LocalStorage 数据持久化

## 核心系统

1. **境界体系** - 炼气→筑基→金丹→元婴→化神→炼虚→合体→大乘→渡劫→仙人
2. **修炼系统** - 挂机自动获取修为/灵气
3. **功法系统** - 主动/被动技能（呼吸吐纳、引气入体、基础剑诀等）
4. **战斗系统** - 野外战斗 + 副本闯关
5. **装备系统** - 武器/防具/饰品
6. **属性系统** - 根骨/悟性/机遇/福源
7. **成就系统** - 20+ 成就等待解锁
8. **离线收益** - 关闭游戏后继续获得修炼收益

## 页面功能

- 🏠 **主页** - 修炼与战斗
- 📜 **功法** - 学习功法
- 🏰 **副本** - 挑战副本与商店
- 🏆 **成就** - 查看成就进度
- 👤 **我的** - 角色信息与设置
